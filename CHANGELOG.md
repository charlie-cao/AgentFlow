# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-12-07

### Added
- Initial release of AgentFlow
- User authentication system with JWT
- Visual workflow editor with React Flow
- AI-powered workflow generation from natural language
- Concurrent Agent execution support
- Real-time execution monitoring via SSE
- Local Ollama integration (qwen3:latest model)
- Optional Supabase integration
- SQLite database support
- RESTful API endpoints
- Workflow execution engine
- Error handling and logging

### Features
- **Authentication**: User registration, login, and token management
- **Workflow Editor**: Drag-and-drop visual workflow creation
- **AI Generation**: Automatic workflow generation from requirements
- **Concurrent Execution**: Parallel execution of multiple agents
- **Real-time Monitoring**: Live execution tracking and debugging
- **Ollama Integration**: Local LLM support with qwen3:latest
- **Supabase Support**: Optional cloud services integration
